# PROGRAMA DE INDICAÇÃO DE PRODUTOS MAROMBAS 



\\ ENTRADA DE DADOS:
    // QUAL O TIPO DE ATIVIDADE QUE VOCE PRATICA? "CHECKBOX"
        + TREINO INTENSO DE CURTA DURAÇÃO {1}
        + TREINO DE LONGA DURAÇÃO {2}
        + TREINO PREDOMINATEMENTE AERÓBICO (CORRIDA) {3}
        + TREINO DE FORÇA E EXPLOSÃO {4}
        + NENHUMA, TENHO PREGUIÇA {5}

    // QUAL SEU OBJETIVO COM OS TREINOS? "CHECKBOX"
        + GANHO DE MASSA MUSCULAR {1 E 4}
        + MANUNTENÇÃO DE MASSA MUSCULAR {1 E 2}
        + MELHORA NA PERFORMANCE {2 E 3}
        + DIMINUIÇÃO DO PERCENTUAL DE GORDURA {1, 2 E 3}
        + FICAR NO SHAPE DEUS GREGO SEM ESFORÇOS {5}

    // O QUE VOCE PROCURA EM UM SUPLEMENTO? "CHECKBOX"
        + MAIS DISPOSIÇÃO PARA TREINAR {2 E 4}
        + AUMENTO NO GANHO DE MASSA {1 E 4}
        + AUMENTO DE PERDA DE GORDURA {1, 2, 3}
        + RECUPERAÇÃO PÓS TREINO MIAS RÁPIDA {1 E 2}
        + MÁGICA MAROMBA {5}

    // O QUE VOCÊ MAIS VALORIZA: "RADIO"
        + UM SHAPE MONSTRO {1, 2, 4}
        + SAÚDE {3 E 4}
        + PERFORMANCE {2 E 4}
        + TRAPÉZIO DESCENDENTE {1, 2, 3, 4}
        + FICAR DEITADO VENTO NETFLIX {5}

    // TEM PRECONCEITO COM A OITAVA MARAVILHA DO MUNDO?? (BOMBAS) "RADIO"
        + CLARO QUE NÃO, NATURAL NÃO CRESCE [PODE RECOMENDAR OS VENENOS]
        + O QUE SÃO BOMBAS?? [TALVEZ RECOMENDA OS VENENOS]
        + SIM, EU SOU UM FRANGO ESQUISITO [NÃO PODE RECOMENDAR OS VENENOS]



\\ LOGICA PERCENTUAL 
    // O NUMERO DE VEZES QUE UM ITEM FOI MARCADO DIVIDIDO PELO NUMERO DE MARCAÇÕES POSSÍVEIS
        + GANHO DE MASSA {1} -> 8 MARCAÇÕES
        + PERFORMANCE {2} -> 8 MARCAÇÕES
        + SECAR {3} -> 5 MARCAÇÕES
        + GANHO DE FORÇA {4} -> 5 MARCAÇÕES
        + PREGUIÇA {5} -> 4 MARCAÇÕES



\\ PROCESSAMENTO DE DADOS:
    // O(S) ITEM(S) MAIS MARCADOS SERÃO INDICADOS AO USUÁRIO
    // CASO O USUÁRIO SEJA IMPARCIAL OU ACEITE TOMAR VENENO O NÚMERO DE SAÍDAS RECOMENDADAS AUMENTA 



\\ SAÍDA DE DADOS:
    // NATURAIS:
        + WHEY (GANHO DE MASSA) {1}
        + PRÉ-TREINO (PERFORMANCE) {2}
        + SHAKE (SECAR) {3}
        + CREATINA (GANHO DE FORÇA) {4}
        + VERGONHA NA CARA (FRANGO) {5}

    // MAROMBAS:
        + HEMOGENIN (GANHO DE MASSA) {1}
        + ESTANOZOLOL (PERFORMANCE) {2}
        + TREMBOLONA (SECAR) {3}
        + OXANDROLONA (GANHO DE FORÇA) {4}
        + CHIFRE (FRANGO) {5}



\\ FUNCIONAMENTO DO PROGRAMA:
    // USUÁRIO FAZ O QUIZ 
        + UMA VARIÁVEL COM O VALOR INICIAL ZERADO PARA CADA POSSIBILIDADE (VAR1 A 5) 
        + ÚLTIMA PERGUNTA VERIFICA SE PODE INDICAR OS "DADOS MAROMBAS"

    // VARIÁVEIS COM VALORES COMPUTADOS 
        + VERIFICAÇÃO PARA SABER QUAL/QUAIS VARIAVEL(IS) TEM MAIOR VALOR 
        + VERIFICAÇÃO PARA SABER SE PODE INDICAR OS "DADOS MAROMBAS"
            - SE FOR TALVEZ, RECOMENDA ALEATÓRIAMENTE
        + EXIBIÇÃO DAS RECOMENDAÇÕES ATRAVÉS DE UM LOOP
